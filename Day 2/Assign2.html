Methods in Array

1.forEach() : This method is used to loop over array's items.

2.includes() : This method check if array includes the item that passed in the method.

3.filter() : This Method create new array with only elements that passed condition inside the function.

4.Array.of() : This create array from every arguments passed into it.

5.Array.from() : This create a array from string by spliting each character in the String into a array.
